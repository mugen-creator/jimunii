# JimuNii - AIバックオフィスアシスタント

## 概要

**JimuNii（ジムニー）**は、LINEで使えるAIバックオフィスアシスタントです。
日常的に使うLINEから、経費精算・スケジュール管理・タスク管理・勤怠記録などのバックオフィス業務を自然な会話で完結できます。

## 機能一覧

### 1. スケジュール管理（Googleカレンダー連携）

| 機能 | 説明 | 使用例 |
|------|------|--------|
| 予定登録 | 日時・場所・メモ付きで予定追加 | 「明日14時に会議」 |
| 繰り返し予定 | 毎週・毎月の定期予定 | 「毎週月曜10時に朝礼」 |
| 予定変更 | 日時・タイトルの変更 | 「さっきの会議、15時に変更」 |
| 予定削除 | 予定のキャンセル | 「明日の会議キャンセル」 |
| 予定確認 | 日/週/月の予定一覧 | 「今週の予定」 |

### 2. 経費精算

| 機能 | 説明 | 使用例 |
|------|------|--------|
| レシートOCR | 撮影だけで金額・店舗・日付を自動認識 | レシート写真を送信 |
| 手動登録 | テキストで簡単登録 | 「タクシー 2400」 |
| 一括登録 | 複数経費をまとめて登録 | 「電車320 昼食850 コピー代150」 |
| カテゴリ自動判定 | 交通費・飲食費など自動分類 | - |
| 誤字自動修正 | 「交遊費」→「交通費」に自動修正 | - |

### 3. タスク管理

| 機能 | 説明 | 使用例 |
|------|------|--------|
| タスク追加 | 優先度・期限付きで追加 | 「請求書作成 今週中」 |
| 一括追加 | 複数タスクをまとめて登録 | 「月末処理」→複数タスク自動生成 |
| タスク完了 | ワンタップで完了マーク | 「請求書作成 完了」 |
| タスク一覧 | 未完了タスクを確認 | 「タスク一覧」 |
| 期限リマインド | 期限前に自動通知 | 毎朝8時 / 夕方5時 |

### 4. 勤怠管理

| 機能 | 説明 | 使用例 |
|------|------|--------|
| 出勤打刻 | LINEで出勤記録 | 「おはよう」「出勤」 |
| 退勤打刻 | 勤務時間自動計算 | 「お疲れ」「退勤」 |
| 勤怠確認 | 今日の勤務状況確認 | 「今日の勤怠」 |

### 5. ファイル管理（Googleドライブ連携）

| 機能 | 説明 | 使用例 |
|------|------|--------|
| ファイル保存 | LINEで送ったファイルをDriveに保存 | PDFを送信→「経理フォルダに保存」 |
| PDF読み取り | PDFのテキスト抽出 | PDF送信で内容確認 |

### 6. リマインダー

| 機能 | 説明 | 使用例 |
|------|------|--------|
| リマインド設定 | 指定日時に通知 | 「明日10時にA社に電話リマインド」 |
| 毎朝通知 | 今日の予定・タスクを毎朝通知 | 自動（8:00） |

### 7. AI機能

| 機能 | 説明 | 使用例 |
|------|------|--------|
| 自然言語理解 | 曖昧な入力を正確に解釈 | 「明後日ミーティング」 |
| 文脈理解 | 「それ」「さっきの」を理解 | 「さっきの取り消し」 |
| 翻訳 | 多言語翻訳 | 「Thank youを日本語に」 |
| メール下書き | ビジネスメール自動生成 | 「A社にお礼メール」 |
| 日報作成 | 日報テンプレート自動生成 | 「日報作成」 |
| 計算 | 簡単な計算 | 「1280×1.1」 |
| 単位変換 | マイル・華氏など | 「100マイルは何キロ」 |

### 8. 名刺管理

| 機能 | 説明 | 使用例 |
|------|------|--------|
| 名刺OCR | 撮影で名前・会社・連絡先を抽出 | 名刺写真を送信 |

### 9. クライアント管理

| 機能 | 説明 | 使用例 |
|------|------|--------|
| クライアント登録 | 取引先情報の登録 | 「A株式会社を登録」 |
| クライアント検索 | 情報の確認 | 「A社の情報」 |
| クライアント一覧 | 全取引先の一覧 | 「クライアント一覧」 |

### 10. 自動ワークフロー

| 機能 | 説明 | タイミング |
|------|------|-----------|
| 月初タスク | 請求書発行、経費集計など自動作成 | 毎月1日 9:00 |
| 月末タスク | 月次レポート、請求確認など自動作成 | 毎月25日 9:00 |
| 週初タスク | 週次ミーティング、週報など自動作成 | 毎週月曜 9:00 |
| 期限通知 | 期限切れ・当日・明日のタスクを通知 | 毎日 8:00 |
| リマインダー | 未完了タスクのリマインド | 毎日 17:00 |

### 11. その他

| 機能 | 説明 | 使用例 |
|------|------|--------|
| 天気予報 | 都市別天気確認 | 「東京の天気」 |
| 定型文管理 | よく使う文章の保存・呼出 | 「挨拶1」で定型文呼出 |
| 音声入力 | 音声メッセージの文字起こし | 音声を送信 |
| グループ対応 | グループLINEで@メンション時のみ応答 | @JimuNii 明日の予定 |

## 技術仕様

| 項目 | 内容 |
|------|------|
| インターフェース | LINE Messaging API |
| AI | Groq (Llama 3.3 70B) |
| OCR | Tesseract.js (日本語対応) |
| 音声認識 | Groq Whisper |
| カレンダー | Google Calendar API |
| ストレージ | Google Drive API / Sheets API |
| ホスティング | Render |
| 言語 | Node.js / Express |
